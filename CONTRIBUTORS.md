## Recognition

Contributors are recognized in the following ways:
- Listed in this file
- Mentioned in release notes for significant contributions
- Credited in commit messages via Co-authored-by

## Becoming a Core Contributor

Core contributors are community members who have made significant and sustained contributions to the project. They have:
- Submitted multiple high-quality pull requests
- Helped with code reviews
- Contributed to documentation
- Assisted other users with issues

If you're interested in becoming a core contributor, please reach out to code@bpluscommunity.org.
